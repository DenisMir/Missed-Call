Missed Call by annoma

The project allows you to support the translation of Missed Call in different languages and you can report issues you have with the application itself.

Welcome to the collaboration repository of Missed Call Missed Call is an application for Android which allows full customization of LED, sound, vibration and repeat notifications for calls (including missed calls), sms, bluetooth events, airplane mode, battery level, package manager and calendar events. In addition it integrates perfectly as a plug-in into Locale for Android. The repository should help us to translate Missed Call into different languages as well as keeping track of feature requests and issues you have with the application.
